# ชุดเอกสารประกวด Knowledge Management

**หัวข้อผลงาน:** การนำ Critical Knowledge ด้านการจัดทำ BOQ และราคากลางงานก่อสร้างท่อร้อยสายสื่อสารใต้ดินไปใช้ผ่านระบบ Conduit BOQ  
**ทีม:** KM/IM Micro Team พัฒนาองค์ความรู้การจัดทำ BOQ และราคากลางงานก่อสร้างท่อร้อยสายสื่อสารใต้ดิน  

---

## 1. วิธีใช้ชุดเอกสารนี้

ชุดเอกสารนี้จัดไว้สำหรับใช้ส่งประกวดงาน Knowledge Management หรือใช้เตรียมรายงาน/สไลด์นำเสนอ โดยวางเรื่องเล่าหลักว่า:

> ทีมได้นำ Critical Knowledge ที่กระจายอยู่ในตัวผู้เชี่ยวชาญ เอกสารเดิม และไฟล์งาน manual มาแลกเปลี่ยนเรียนรู้ จัดระบบ ตรวจสอบ และแปลงเป็นระบบดิจิทัล Conduit BOQ พร้อมฐานข้อมูลราคากลาง workflow มาตรฐาน และหลักฐานการใช้งานจริงที่วัดผลได้

## 6. Checklist ก่อนส่งประกวด

### ข้อมูลที่ควรเติมเพิ่ม

- [ ] ชื่อผู้สนับสนุนทีม
- [ ] ชื่อหัวหน้าทีม
- [ ] ชื่อ KM Agent
- [ ] รายชื่อสมาชิกทีม
- [ ] รูปกิจกรรม KM / workshop
- [ ] รายชื่อผู้เข้าร่วมกิจกรรม
- [ ] screenshot ระบบ dashboard / BOQ edit / price list / print / export
- [ ] ตัวอย่าง BOQ output ที่ไม่เปิดเผยข้อมูลลับ
- [ ] ผล time study หรือ survey ความพึงพอใจ ถ้ามี

### จุดที่ควรเล่าให้ชัดกับกรรมการ

- [ ] Critical Knowledge คืออะไร และเสี่ยงสูญหายอย่างไร
- [ ] ทีมแลกเปลี่ยนเรียนรู้อย่างไร
- [ ] Tacit Knowledge ถูกแปลงเป็น Explicit Knowledge อย่างไร
- [ ] ความรู้ถูกนำไปใช้จริงใน workflow ใด
- [ ] มีหลักฐานตัวเลขอะไรจาก production database
- [ ] ผลงานต่อยอดได้อย่างไรในระดับองค์กร

## 7. One-Minute Pitch

ทีม KM/IM Micro Team ได้นำ Critical Knowledge ด้านการจัดทำ BOQ และราคากลางงานก่อสร้างท่อร้อยสายสื่อสารใต้ดิน ซึ่งเดิมกระจายอยู่ในผู้เชี่ยวชาญ เอกสาร และไฟล์ manual มารวบรวม ตรวจสอบ จัดระบบ และแปลงเป็นระบบ Conduit BOQ ทำให้เกิด price list กลาง 710 รายการ workflow จัดทำ BOQ หลายเส้นทาง การคำนวณ Factor F/VAT อัตโนมัติ และข้อมูล production ที่วัดผลได้จริง เช่น BOQ 187 รายการ และ BOQ items 1,475 รายการ ผลงานนี้จึงเป็นการจัดการความรู้ที่นำไปใช้จริง วัดผลได้ ถ่ายทอดได้ และต่อยอดเป็นมาตรฐานองค์กรได้
